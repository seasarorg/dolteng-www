Doltengがサポートするフレームワークの組み合わせ

概要

Doltengは多数のフレームワークをサポートしていますが，全ての組み合わせでテストされているわけではなく，組み合わせによって使えない機能もあります．

新規作成Wizard

新規作成Wizardで選択できる組み合わせのうち，テストされている組み合わせは以下のとおりです．表にない組み合わせは，最新のリリースではテストされていません．問題を発見した場合はメーリングリスト等でお知らせください．

Application Type Presentation Persistence S2JMS (Outbound) S2RMI (Client) ServerManagement JRE
Web Application Teeda (Page only) - - - Sysdeo Tomcat Plugin 1.5
S2Dao - - Sysdeo Tomcat Plugin 1.5
Kuina-Dao (Hibernate) - - Sysdeo Tomcat Plugin 1.5
○ - Sysdeo Tomcat Plugin 1.5
- ○ Sysdeo Tomcat Plugin 1.5
SAStruts S2JDBC - - Sysdeo Tomcat Plugin 1.5
- - WTP (Servlet2.4) 1.5
S2Flex2 (with Flex3.5) S2Dao - - Sysdeo Tomcat Plugin 1.5
S2JDBC - - Sysdeo Tomcat Plugin 1.5
S2BlazeDS (Server Only with BlazeDS 3) S2Dao - - Sysdeo Tomcat Plugin 1.5

S2BlazeDS (Server Only with BlazeDS4) S2Dao - - Sysdeo Tomcat Plugin 1.5
